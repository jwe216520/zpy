第 5 章　植物體的生長發育

選修生物 II ── 重點整理

5-1
植物體的生長發育受激素調節
植物激素概述
定義微量有機物，調節植物的生長發育
產生部位個體中所有活細胞皆能產生，各激素產生部位可能不同
運送方式藉由擴散或維管束組織運送
作用特性微量即可作用；需與目標細胞受體結合；同種激素在不同細胞可能功能不同；不同種激素彼此間可能有交互作用（協同或拮抗）
五大植物激素比較
激素 縮寫 產生部位 主要功能
生長素 IAA 根尖與莖頂分生組織、嫩芽、幼葉及發育中種子 促進細胞延伸與組織分化；與向性有關；頂芽優勢；農業（扦插、嫁接、除草劑 2,4-D）
吉貝素 GA 根尖與莖頂分生組織、嫩芽、幼葉及種子內的胚 促進節間延長；促進穀類種子萌發；打破休眠；促進長日照植物開花（取代春化作用）
細胞分裂素 CK 根部合成後運至地上部；覆盆子等果實合成 促進細胞分裂；延遲葉片老化；促進腋芽生長；抑制頂芽優勢（與 IAA 拮抗）
離層酸 ABA 植物體大部分細胞、未成熟或逆境中的成熟種子 促進芽與種子休眠；促進氣孔關閉（抗旱）；促進葉片老化（逆境激素）
乙烯 C₂H₄ 植物大部分細胞（氣態，易擴散） 促進果實成熟；抑制細胞延長；促進葉老化與離層形成；促進淹水時根部細胞計畫性死亡（形成氣室）
生長素（IAA）詳解 IAA
發現者達爾文（Darwin）父子 ── 芽鞘向光性研究
天然種類吲哚乙酸（IAA）；人工合成：萘乙酸（NAA）、二氯苯氧乙酸（2,4-D）

功能 1：向性（生長反應）

  • 向光性：側光照射 → IAA 由向光側移向背光側 → 背光側 IAA 多 → 背光側細胞伸長快 → 莖向光彎曲
  • 背地性（向性）：IAA 在向地側含量多 → 向地側細胞伸長多 → 莖背地生長
  • 不同器官對 IAA 的感受性不同：莖 > 芽 > 根；同濃度對根可能抑制，對莖卻促進

功能 2：頂芽優勢

  • 莖頂 IAA 濃度高 → 促進頂芽生長，抑制鄰近側芽（保持休眠）
  • 去除頂芽 → 側芽生長明顯；以 NAA 塗於切口處 → 側芽僅微生長

農業應用

  • 刺激形成層細胞分裂 → 提高嫁接成功率
  • 促進不定根形成 → 提高扦插繁殖成功率
  • 促進果實肥大（如草莓種子產生 IAA，促進花托生長）
  • 抑制離層產生 → 維持果柄生長，延遲落果
  • 2,4-D 為除草劑：去除雙子葉雜草，保全單子葉農作物
  • 誘導單性結果（雌蕊不授粉，直接發育出果實）
吉貝素（GA）詳解 GA
發現來源感染水稻的Gibberella fujikuroi（惡苗病菌）

主要功能

  • 促進節間延長：刺激細胞分裂與延長（矮莖植物施 GA 可恢復正常高度，如捲心甘藍）
  • 促進種子萌發：種子吸水 → 胚產生 GA → GA 進入胚乳 → 促進糊粉層分泌水解酶 → 分解澱粉等大分子 → 小分子養分供幼苗生長
  • 促進長日照植物開花（取代春化作用，如彩葉甘藍）
  • 促進雄花分化（部分植物）
細胞分裂素（CK）詳解 CK
發現者史庫格（Skoog）── 由酵母菌發現
  • 促進細胞分裂；延遲組織老化（延緩葉片黃化）
  • 促進腋芽生長（側芽生長）
  • 抑制根部氣孔關閉（對 ABA 的拮抗）
  • 促進花芽分化（洋桔梗、蘋果、荔枝等）
生長素與細胞分裂素的組織培養應用：
IAA/CK 比值高 → 促進根分化；
IAA/CK 比值低（接近）→ 促進莖葉分化；
IAA/CK 比值居中 → 形成癒傷組織
離層酸（ABA）詳解 ABA
別稱休眠素；逆境激素
形成離層造成葉、花、果的自然脫落（後來研究發現造成離層的主因是乙烯，非 ABA）
  • 促進芽與種子休眠：未成熟種子中高濃度 ABA 抑制萌發；成熟種子中可使種子休眠
  • 促進氣孔關閉：植物嚴重缺水時，根部產生 ABA 運至葉片 → 氣孔關閉 → 減少水分蒸散
  • 與 GA 拮抗：種子打破休眠後，ABA 濃度下降，GA 濃度升高，促進萌發
  • ABA 也能抑制莖伸長（與 IAA 有拮抗作用）
乙烯（Ethylene）詳解 C₂H₄
特性氣態植物激素，易擴散到植物各部位
  • 促進果實成熟：使未成熟的奇異果、木瓜的顏色和氣味發生變化（更甜、更軟）
  • 抑制細胞延長：抑制幼苗胚軸伸長速率，使胚軸粗短，助於茁壯
  • 細胞計畫性死亡：長期淹水缺氧 → 乙烯誘導部分根的細胞計畫性死亡 → 增加空隙 → 利於氣體交換
  • 促進葉老化與離層形成（實為離層形成的主因）
乙烯與 IAA 的交互作用（葉脫落過程）：
幼葉期：IAA 含量高 → 降低葉柄對乙烯的敏感度 → 避免離層形成
老葉期：葉不再生成 IAA → 乙烯增加 → 加速老化 → 誘導葉柄基部形成離層 → 老葉脫落
激素交互作用總整理
效應協同（+）激素拮抗（−）激素
篂縩（細胞分裂）+ IAA、CK
莖的伸長+ IAA、GA− ABA
種子休眠+ ABA− GA
氣孔關閉+ ABA− IAA（根）、CK
葉片老化+ 乙烯− CK
腋芽生長+ CK− IAA（頂芽優勢）
篂縩脛ⵋ（組織老化延遲）+ CK− 乙烯
5-2
植物體對環境刺激的反應
向性（Tropism）
向性影響因子機制簡介
向光性 側光 頂芽產生的 IAA 向下運送時，側光造成 IAA 不均勻分布，背光側 IAA 多 → 莖向光彎曲
向地性 重力 根冠內平衡細胞含澱粉粒，可感知重力 → 影響 IAA 分布 → 造成根的向地性
向觸性 接觸物體或枝條 植物對機械性刺激，產生細胞伸長不對稱 → 造成向觸性（如藤蔓纏繞）
傾性（Nastic Movement）── 非向性
  • 植物因膨壓變化而發生的可逆性運動，不受刺激方向影響
  • 含羞草的觸發運動（葉枕膨壓變化 → 小葉向上閉合）
  • 捕蠅草的捕蟲運動
  • 酢漿草在太陽下山後葉片下垂
  • 特性：具可逆性（可恢復原狀）
光週期性（Photoperiodism）── 光對開花的影響

短日照植物

需連續黑暗期夠長（大於臨界夜長）才能開花
例：菊花、水稻
臨界夜長約 9 小時以上

長日照植物

需連續黑暗期夠短（小於臨界夜長）才能開花
例：鳶尾、菠菜
臨界夜長約 14 小時以下

中性植物

開花不受光週期影響
例：番茄、向日葵
在任何光週期下均可開花

關鍵：決定開花的是「連續黑暗期的長短」而非日照長短！
在黑暗期中斷（以紅光照射）→ 短日照植物不開花，長日照植物開花。

光敏素（Phytochrome）機制

Pr（紅光型）
── 紅光（660 nm）→
Pfr（遠紅光型）
活性態
Pfr
── 遠紅光（730 nm）或黑暗→
Pr
  • 光敏素存在於袋狀體（覆膜小泡）與葉綠體中
  • Pfr/Ptotal 比例高 → 促進長日照植物開花；抑制短日照植物開花
  • Pfr 促進種子萌發、莖葉伸長、光合作用色素合成
環境對種子萌發的影響
環境因子影響說明
水分 種子吸水後，激活種子內酵素，代謝加速，促進萌發
溫度 適溫加速酵素反應；春化作用（需一段低溫期才能萌發，如冬小麥）
光照 需光種子：Pfr 嬡❛高→促進萌發（如萵苣）；嫌光種子：Pr 比例高 → 促進萌發
氧氣 種子行有氧呼吸以供應萌發所需能量，缺氧則抑制萌發
春化作用（Vernalization）：需要一段低溫（0～10°C）才能開花結實的現象。
例：冬小麥需在冬季低溫中生長，隔年春天才能抽穗開花。
感受低溫的部位：莖頂與胚（分生組織）
逆境對植物生長發育的影響
逆境類型植物的應對機制
缺水逆境 ABA 增加 → 氣孔關閉（減少水分蒸散）；ABA 抑制細胞生長；乙烯增加（促進落葉）
高鹽逆境 乙烯增加 → 促進氣孔關閉；調節細胞滲透壓（積累相容性溶質）；部分植物以囊泡分泌鹽分
淹水逆境 乙烯大量合成 → 誘導部分根的細胞計畫性死亡 → 形成氣室（空隙）→ 利於氣體交換（如玉米）
蟲害逆境 立即性防禦：觸發過敏反應，生成抵抗入侵化學物質
局部性防禦：細胞壁木質化、細胞計畫性死亡、茉莉酸誘導防禦蛋白
系統性防禦：茉莉酸及衍生揮發物（水楊酸、水楊酸甲酯）經篩管或空氣傳遞至未受害部位
探討活動 5-1
發現植物的生長素 ── 實驗設計與結果解讀
達爾文父子（1880）
問題芽鞘為何會向光生長？
假說芽鞘頂端會感應側光而向光生長
結論芽鞘頂端感應光，並傳遞訊號給下方使不均勻生長
柏伊森-詹生（1913）
材料雲母片（不透水）、洋菜膠（可透水）
結論芽鞘頂端感應光的物質可經洋菜膠傳遞（化學性訊號），無法穿過雲母片
鮑爾（1918）
操作將切除的芽鞘頂端置於芽鞘一側，置於黑暗中
結論芽鞘頂端可產生促進生長的化學物質（不需光照，只要不均勻分布即可彎曲）
文特（1926）
操作用洋菜膠吸收頂端物質後，置於去頂芽鞘一側，黑暗中觀察彎曲角度
結論物質濃度越多，彎曲角度越大；命名此物質為「生長素」
